# gkbot-marathon

此為練習用 `Vue3` 開發的純前端專案，採用 `Composition API` coding style，部分資料儲存串接 `Google App Scripts`，以 `Google Spreadsheets` 作為資料儲存。

## 技術棧
- `Vue3`
- `Vue3 Composition API`
- `Vuex`：狀態管理
- `TypeScript`
- `Google App Scripts`：遊玩紀錄資料儲存
